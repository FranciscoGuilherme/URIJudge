# Exercício 1003

> Categoria iniciante

- [Requisitos](#requisitos)

## <span id='requisitos'>Requisitos</span>

- Instalar o utilitário _make_ em seu sistema

## <span id='executavel'>Gerando o executável</span>

- Acessar o diretório _build_
- Executar o comando
    ``` sh
    $ make build.program
    ```
